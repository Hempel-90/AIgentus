# AIgentus

## Beschreibung

AIgentus ist ein fiktiver KI-Agent der fiktiven Firma ArBytes.Amt, die im Rahmen der Umschulung entwickelt wurde. Ziel war die Umsetzung einer modernen Webanwendung mit dynamischen Inhalten und Datenbankanbindung.

## Funktionen

- Mehrseitige Unternehmenswebsite
- Dynamische Inhalte
- Datenbankanbindung
- Benutzerfreundliche Navigation
- Responsives Layout

## Verwendete Technologien

- PHP
- Laravel
- HTML
- CSS
- MariaDB

## Projektziel

Praktische Anwendung moderner Webentwicklung mit Laravel und relationalen Datenbanken.

## Status

Praxisprojekt im Rahmen der Umschulung zum Fachinformatiker für Anwendungsentwicklung.
